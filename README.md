# AirSense

AirSense is a full-stack smart air quality micro-monitoring dashboard with:

- Hyperlocal AQI map view
- Trend analysis with simple 6-hour prediction
- Real-time data refresh
- Alerts for unhealthy conditions and sensor issues

## Tech Stack

- Frontend: React (Vite), Tailwind CSS, React Leaflet, Recharts, Remix Icons
- Backend: Node.js, Express,MongoDb,nodes corn

## Project Structure

- `backend/` - Express API server
- `frontend/` - React dashboard client


- `GET /api/trends` - historical + predicted AQI trend data
